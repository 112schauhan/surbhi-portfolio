# Surbhi Singh – Portfolio

Personal portfolio site showcasing experience, projects, skills, and contact information.

**Live site:** [https://surbhi-portfolio-rho.vercel.app/](https://surbhi-portfolio-rho.vercel.app/)

## Tech stack

- **React** + **TypeScript** + **Vite**
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **tsparticles** for the particle background
- **React Icons** (Contact section)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Deploy

The app is set up for deployment on [Vercel](https://vercel.com). The live site is hosted at [https://surbhi-portfolio-rho.vercel.app/](https://surbhi-portfolio-rho.vercel.app/).
